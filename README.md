# docker_bash

#### 介绍
编排了一些常用的docker镜像，脚本中的容器是测试案例，用于参考

#### 软件架构
必须先安装docker和docker-compose

#### 使用前须知
需要将`main.sh`和`docker-compose.yml`中的`flag`改为自定义的项目名称。

#### 支持镜像
mongo       略  
mysql       略  
mariadb     略  
redis       略  
java        略  
nginx       略  
ipfs        星际网络存储协议  
portainer   docker可视化管理  
mq          消息队列  
yapi        接口管理
